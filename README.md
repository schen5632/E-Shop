Fully responsive full-stack ecommerce web application developed using Java, Spring Boot, React, NodeJS, Redux, MySQL, and Stripe. Deployed backend and frontend to AWS using AWS EC2, Elastic Beanstalk, S3, RDS. 

Link for web application hosted on AWS: http://eshop-frontend.s3-website.ca-central-1.amazonaws.com/
